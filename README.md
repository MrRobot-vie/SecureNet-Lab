# SecureNet Lab

Infrastructure d'entreprise simulée avec des outils pour l'administrer et la sécuriser.

## Modules
- Module 1 : Scanner réseau
- Module 2 : Gestionnaire SSH
- Module 3 : Surveillance réseau
- Module 4 : SIEM simplifié
- Module 5 : Gestionnaire de certificats
- Module 6 : Gestionnaire de mots de passe
- Module 7 : Tableau de bord web

## Technologies
- Python, Java
- VirtualBox, Kali Linux, Ubuntu Server
- SQLite, PostgreSQL, Flask
